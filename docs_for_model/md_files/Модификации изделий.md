# Модификация изделия

**Модификация изделия** — это разновидность товара, которая отличается от других вариантов одной и той же модели по одной или нескольким характеристикам, таким как размер, цвет, материал или комплектация. Документ можно создать двумя способами:

1. Документ можно найти через поиск в главном меню системы в модуле "Управление производством"

![](image.png)

После нажатия на **"Модификации изделий"** откроется реестр документов. 

![](image-1.png)

Для создания документа нужно нажать на кнопку ![](image-2.png)

Откроется форма документа

![](image-3.png)

Документ состоит из двух частей: Шапки и табличной части.

В шапке документа автоматически заполняются такие поля, как:

* **Организация** - Заполняется той, в которой вы находитесь
* **Дата создания** - Заполняется текущей датой
* **Статус** - По умолчанию заполняется значением "Создан"
* **Номер документа** - Заполняется автоматически при сохранении документа
* **Период (с/по)** - Заполняется вручную

Для заполнения табличной части нужно нажать на кнопку ![](image-4.png)

Откроется окно выбора заявок от клиента, в котором нужно будет выделить галочками нужные нам заявки и нажать на кнопку **"Добавить в план"**.

![](image-5.png)

После этого в табличной части появятся выделенные строки

![](image-6.png)

Для того, что бы перейти к заявке нужно нажать на нее в колонке **"Заявка"**

![](image-7.png)

По кнопке ![](image-8.png) можно разузловать позиции, выбрав их предварительно галочками.

![](image-9.png)

В табличной части поменяется "Статус раузлования" и подсветится зеленым цветом.

Во вкладке **"Разузлование"** можно посмотреть из чего состоит изделие. Так же есть возможность из табличной части перейти в Технологическую карту материалов с видом ДСЕ и в Заявку от клиента.

![](image-10.png)

По кнопке ![](image-11.png) можно создать Заявку на потребность на материалы с видом "Покупной" предварительно выбрав материал нажав на него.

Откроется форма создания Заявки на потребность 

![](image-12.png)

Кнопками ![](image-13.png) можно удалить или добавить номенклатуру в узел.

Во вкладке "Расчет баланса времени" можно увидеть информацию о времени, которое понадобится для изготовления определенного узла. Для этого нужно нажать на соответствующую кнопку.

![](image-14.png)
