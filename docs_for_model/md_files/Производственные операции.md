# Производственные операции

Документ предназначен для учета операций, совершаемых при изготовлении готовой продукции.
Для работы с документом **Производственные операции** необходимо перейти в **Меню** – найти группу **Управление производством** – открыть документ **Производственные операции**.

![](image.png)

Нажмите на **Производственные операции**, откроется реестр. Для того, чтобы открыть ранее созданный документ требуется нажать на него. Для того, чтобы создать новый документ нажмите на кнопку **"Создать"**.

![](image-1.png)

Откроется новый документ, который состоит из двух вкладок: Основная и Подразделения.

![](image-2.png)

На вкладке **"Основная"** нужно заполнить такие поля, как:

* **Наименование** – обязательное текстовое поле документа, которое прописывается вручную. 
* **Родитель в обозревателе** –  обязательное поле, заполняется автоматически. 
* **Вид операции** – поле, данные которого выбираются из выпадающего списка. 
* **Содержание** – текстовое поле, информация прописывается вручную. По желанию можно оставить пустым. 
* **Период действия: с/по** – заполняется вручную или путем выбора в календаре. 
* **Добавить** – поле, данные которого выбираются из списка.

После выбора из “Добавить”, под данной строкой появляется таблица. Данные заполняются из выпадающего списка, чекбоксы отмечаются галочкой.

![](image-3.png)

На вкладке **“Подразделения”** вы можете заполнить информацию о подразделении и периоде действия.

![](image-4.png)

Для того, чтобы **Сохранить**, **Сохр.закрыть** или **Закрыть** документ, необходимо нажать на кнопки, которые находятся в шапке документа:

![](image-5.png)
