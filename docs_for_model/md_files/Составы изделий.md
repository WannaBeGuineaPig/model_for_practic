# Составы изделий

Состав изделия представляет собой перечень материалов и других составляющих, используемых при производстве изделия. Все описания материалов, полуфабрикатов, комплектующих и готовых изделий хранятся в информационной базе в общем номенклатурном справочнике, что упрощает процесс составления спецификаций и, в конечном итоге, облегчает планирование и управление. Комплектующее или готовое изделие может состоять из любых других элементов списка номенклатуры. В свою очередь, оно может быть включено в состав другого изделия. 

Для работы с документом **Составы изделий** необходимо перейти в **Меню** – найти группу **Управление производством** – открыть документ **Составы изделий**.

![](image.png)

Нажмите на **Составы изделий**, откроется реестр. Чтобы открыть ранее созданный документ требуется нажать на него. Чтобы создать новый документ, требуется нажать на кнопку **Создать**

![](1.jpg)

Откроется новый документ

![](image-1.png)

Поля, обязательные для заполнения:

- **Номер** - заполняется автоматически при сохранения документа или прописывается вручную;
- **Изделие** – заполняется 2 способами: выбором из списка (![](3.jpg)) или из обозревателя аналитик (![](4-1.jpg)). После выбора изделия можно перейти в него и внести правки (кнопка ![](5-1.jpg)). Подробнее о заполнении справочника номенклатур [читать тут](Lexema-ERP/../../../Общие%20справочники/Справочники.md#номенклатура);

- **Выход ГП и Ед. измерения** – заполняется автоматически после выбора изделия. По необходимости можно выбрать альтернативную ед.измерения , если она указана в справочнике номенклатур. 

Поля, которые заполняются по умолчанию:

- **Дата** - по умолчанию заполняется текущей датой;

- **Статус** - по умолчанию заполняется значением “Создано”;

- **Шифр модификации** – заполняется автоматически после выбора Изделия.

Текстовые поля, которые заполняются по необходимости:

- **Номер редакции, номер извещения** – номера, которые уникальным образом идентифицируют Состав изделия;

- **Примечание** – вводится текстовое сообщение. 

Есть возможность выбрать Базовый СИ (если есть) - выбирается из списка. При нажатии на ![](5-2.jpg) осуществится переход в Базовый СИ:

![](image-2.png)

При нажатии на кнопку **“Заполнить”** заполнится таблица значениями из Базового СИ:

![](image-3.png)

При необходимости можно заполнить такие поля как: Контрагент и Заявка от клиента.

Таблица “Спецификация” также заполняется по кнопке ![](10-1.jpg). В новой строке выберите материал из списка, обозревателя или создайте новый. Не забудьте указать количество:

![](11-1.jpg)

Кнопка **"Входимость узла"** предназначена для просмотра изделий из которых состоит изделие в создаваемом документе. Чтобы посмотреть входимость узла выберите одну позицию из Спецификации, а затем нажмите на кнопку “Входимость узла”. Откроется окно, в котором отражается Количество, Родительский узел а также документ:

![](12-1.jpg)

Для того, чтобы на основании созданного Состава изделия создать Технологическую карту , поменяйте статус документа на “Проверено” и по кнопке **"Операции"** выбрать Создать технологическую карту.

![](image-4.png)

Документ СИ можно копировать, вкладывать вложения, отправлять по маршруту. Для этого используются соответствующие кнопки на панели управления.
