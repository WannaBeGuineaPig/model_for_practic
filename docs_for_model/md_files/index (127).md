# Пользовательские списки рассылки

## Описание

Данный документ предназначен для формирования пользовательского списка рассылки, например, руководителю подразделения необходимо ознакомить своих подчиненных с локальным нормативным актом по предприятию. Указанным пользователям из этого списка данный документ будет направлен на ознакомление.

## Описание функционала документа

Для создания новой формы в меню поиска необходимо ввести "Пользовательские списки рассылки" и перейти по найденному результату. В открывшемся реестре  создания нового списка в панели инструментов необходимо нажать кнопку"Создать".

Откроется новая форма.

* Поле **"Наименование"** группы рассылки заполняется пользователем (поле текстовое).

* **Владелец** автоматически заполняется текущим пользователем. Пользователь обладающий специальными правами может 

* **Тип документа** - из выпадающего списка выбирается тип документа, к которому будет распространяться данный список рассылки.

Например, если выбран тип документа "Локальный нормативный документ", то эту группу рассылки можно будет выбрать в окне список рассылок документа "Локальный нормативный документ".

Тип документа может содержать несколько выбранных вариантов.
