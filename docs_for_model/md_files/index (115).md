# Работа с обращениями

## Описание

Форма "Работа с обращениями" нужна для регистрации и обработки обращений из внешних источников, например, обращение от пациентов медицинского учреждения по качеству получения услуг.

## Работа с формой

Для создания нового обращения необходимо перейти в меню и ввести "Карточка обращения".
