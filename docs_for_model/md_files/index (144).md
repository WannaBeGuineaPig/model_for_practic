# Машиночитаемая доверенность

Для открытия документа необходимо в Меню в поисковой строке ввести - "Доверенности" и открыть эту форму

Откроется реестр документов:

![Рисунок 2 Реестр документов "Доверенности на конструкторе](./media/image2.png)

В колонке **"Подписан ЭП"** отображается зелёный значок, указывающий на наличие подписи КЭП на завершенном этапе маршрута.

С помощью кнопки **"Загрузить из Контура"** в реестре загружается МЧД из системы Контур.Доверенности.

Для создания нового документа нажмите **"Создать"**. Откроется форма для заполнения:

![Alt text](media/image.png)

Для начала работы с документом необходимо выбрать из справочника Подтип документа - **Электронная машиночитаемая доверенность**.

![Alt text](media/image-1.png)

В блоке "МЧД" заполняется специфичная информация о МЧД.

**Вид** - вид электронной доверенности. Для организаций - B2B, для государственных учреждений B2G

**Формат** выбирается из выпадающего списка формат МЧД. Он определяет структуру и спецификации, согласно которым будет создана машиночитаемая доверенность.

**Историческая** - проставляется автоматически только у тех доверенностей, которые были загружены из других интеграционных систем.

**Статус в реестре ФНС** и **Id доверенности** - проставляется автоматически только у тех доверенностей, которые были загружены в другие интеграционные системы после процесса согласования.

Для доверенностей, загруженных из внешних систем, указанные выше поля заполняются автоматически.

![Alt text](media/mchd2.png)

Необходимо также заполнить атрибуты **Период действия "с" и "по"** доверенности.

![Рисунок](media/screen3.png)

**Вид** доверенности для государственных учреждений B2G имеет поле "Вид налогового органа". Это поле обязательно для заполнения при формировании документа. 
 
## Работа с документом

После заполнения всех необходимых сведений необходимо нажать на кнопку "Заполнить шаблон". 

Далее необходимо отправить документ по маршруту, нажав на кнопку "Маршрут" на панели инструментов. Более подробно работа с маршрута описана в разделе [Редактирование маршрута](../../Работа с маршрутами/).

Можно настроить направление уведомления-напоминания инициатору и поверенным об окончании срока действия доверенности за указанное количество дней до истечения с предложением сделать заявку на выдачу новой доверенности. Для этого необходимо обратиться к администратору сайта.
