# Система скидок

Документ предназначен для введения скидок потенциальным заказчикам. 

## Реестр

Для открытия реестра документов необходимо зайти в **Меню** – Найти группу **Складской учет** – Открыть документ **Система скидок**.

![](1.png)

Или ввести в поисковое Меню **”Система скидок”**.

![](2.png)

Откроется реестр системы скидок:

![](1-1.png)

## Создание документа 

В левом верхнем углу реестра нажмите кнопку **“Создать”**.  

![](1-2.png)

Откроется новый документ:

![](5.png)

!!! note "Примечание"
     Поля, отмеченные ”*”, обязательны для заполнения

- **”Номер”** заполняется автоматически при сохранении документа. Есть возможность прописать вручную. 

- **”Период с”** – обязательное для заполнения поле. Дата встает автоматически, но внести изменения можно. Для этого следует прописать ее вручную или внести при помощи календаря:

![](6.png)

- **”Период по”** – поле, которое можно оставить пустым. Оно указывает дату окончания действия скидки. 

- **”Статус”** – обязательное для заполнения поле. При создании документа автоматически проставляется значением “Создано”. Сотрудник имеет право менять статус, в зависимость от необходимой цели. Если ему нужно, чтобы скидка начала действовать, он переводит документ в статус “Утверждено”. Если же он хочет, чтобы скидки перестали действовать, он присваивает документу статус “Архив”.

![](7.png)

- **”Ценовая группа”** – по умолчанию встает Фиксированная цена. Выбрать другую группу можно при помощи выпадающего списка. Эти группы заводятся в справочнике “Ценовые группы”. 

![](8.png)

По кнопке **”+”** в талице “Детализация” добавляется новая строка. Она нужно для добавления номенклатуры. Автоматически встало значение “Отпускная цена” в поле “Вид цены прайса”. 

![](9.png)

В поле “Код и краткое наименование номенклатуры” выбираем необходимую позицию. Отобразились цены по прайсу:

![](10.png)

“Вид цены прайса” по желанию можно изменить:

![](11.png)

Оставим отпускную цену и проставим скидку 10%: 

![](12.png)

Теперь Цена с НДС и Цена без НДС уменьшилась на 10%.

![](13.png)

Повторим шаги с двумя другими номенклатурами:

![](14.png)

Вкладка “Покупатели” предназначена для выбора заказчиков:

![](15.png)

Выбранные покупатели отобразятся в шапке документа:

![](16.png)

Для того, **чтобы была возможность применять систему скидок, необходимо присвоить документу статус “Утверждено”**, а затем нажать на “Сохранить” или “Сох.Закрыть”.

![](17.png)
