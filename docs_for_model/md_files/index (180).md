# Авторизация в мобильном приложении

Для входа в приложение введите "Имя пользователя", "Пароль" и нажмите "Войти"

![Авторизация в мобильном приложении](media/image1.png)

## Двухфакторная авторизация

В случае если на проекте используется двухфакторная авторизация, то после ввода логина и пароля откроется новое окно, в котором необходимо вести код отправленный через каналы связи: электронная почта, СМС, или посредством входящего вызова на смартфон.

![Почта](media/mail.png)
![СМС](media/sms.png)
![Входящий вызов](media/call.png)

## Пин-код для входа

Чтобы при авторизации каждый раз не вводить пароль, в приложении возможно задать пин-код. Для авторизации в приложении достаточно ввести пин-код.

![Пин-код](media/pinCode.png)

!!! Примечание
    На ввод пин-кода при авторизации дается 4 попытки. По окончании попыток  необходимо авторизоваться повторно.

    ![Авторизация](media/autoriz.png)
