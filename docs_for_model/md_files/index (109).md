# Связка документов

## Описание

Связка между документами в документообороте представляет собой обычно ссылку или связь между двумя или более документами, которые которые связаны по содержанию, теме, контексту или каким-либо другим параметрам. В случаях когда один документ создается на основе другого, например, при получении входящего письма создается в ответ исходящее, в данном случае связка между документами производится автоматически.

## Ручное связывание документов

Для того чтобы связать документы между собой необходимо сперва перейти в один из документов, которые необходимо связать между собой, в панели инструментов нажать на кнопку **"Связки"** и нажать **"Добавить текущий"**. Текущий документ появится в блоке **"Претенденты"**.

![alt text](media/image-1.png)

Далее необходимо перейти в другой документ который необходимо связать с текущим. В случае необходимости можно добавить более одного документа для последующих связок.

![alt text](media/image-2.png)

Если документ был добавлен случайно, необходимо его выделить и нажать кнопку **"Удалить"**.

Далее находясь в текущем документе, в блоке **"Претенденты"** выбрать документы которые необходимо связать и нажать соответствующую кнопку.

![alt text](media/image-3.png)

Связка между документами будет отображаться в блоке **"Связанные документы"**. 

![alt text](media/image-4.png)
