# План производства 

**План производства** — это стратегический документ, который определяет, какие товары или услуги будут производиться, в каком объеме и в какие сроки. Он помогает организовать эффективное использование ресурсов, минимизировать затраты и риски, а также обеспечить своевременное выполнение заказов и удовлетворение потребностей клиентов. Он является важным инструментом для управления производственными процессами и принятия обоснованных решений.

Чтобы создать план производства, необходимо в главном меню перейти в **"План производства"**. 

![](image.png)

Откроется реестр имеющихся документов. 

![](image-1.png)

Для того, чтобы открыть старый документ - кликните по нему, для того, чтобы создать новый - нажмите по соответствующей кнопке ![](image-2.png)

Откроется форма создания нового документа, состоящая из шапки документа и табличной части 

![](image-3.png)

В шапку документа автоматически заполнятся такие поля, как:

- **Организация** – заполнена организацией, находясь в которой пользователь формирует план.
- **Дата создания** - заполняется текущей датой.
- **Подразделение** - заполняется подразделением, указанным в вашем сотруднике. 
- **Статус** – по умолчанию новый документ создается в статусе «Создан». После утверждения плана сменить на **«Утвержден»**.
- **Номер** - заполняется автоматически при сохранении документа.

Поля для самостоятельного заполнения:

- **Период с/по** – указать период, в течении которого будет реализовываться план.

- **Группа номенклатуры** - если в выпадающем списке выбрать какую либо группу, то в загрузке заявок от клиента будут только те заявки, в которых есть эта группа номенклатуры.

![](image-4.png)

Для того, чтобы заполнить табличную часть нужно на вкладке **План** нажать на кнопку **Загрузить**

![](image-5.png)

По данной кнопке откроется окно выбора заявок от клиента, на основе которых будет сформирован план. 

!!! Примечание
    Чтобы в плане производства нашлась заявка от клиента, она должна быть в статусе «Подтвержден».

Нужно галочкой выделить необходимые заявки и нажать кнопку **Добавить в план производства**. 

![](image-7.png)

В табличной части заполнятся значения из выбранных заявок в колонках **Номенклатура**, **Количество**, а также информация о заказчике, номер заявки и др. заполнится номенклатурой из выбранных заявок от клиента.

![](image-6.png)

Далее требуется провести разузлование номенклатуры в соответствии с выбранной технологической картой. 

!!! Примечание
    По умолчанию разузлование происходит в соответствии с составом изделия, с помощью константы **disassembly_for_production_plan_by_production_card** разузлование происходит в соответствии с технологической картой и она должна быть со статусом «Проверено».
Для этого выделите галочкой номенклатуру и нажмите кнопку **Разузловать**. 

![](image-8.png)

Вы автоматически перейдете на вкладку **"Разузлование"**. В этой вкладке можно перейти в тех.карту и заявку от клиента кликнув по соответствующей колонке.

![](image-10.png)
 
На вкладке "План" в колонке **Статус разузлования** статус сменится на **Совпадает**. 

![](image-9.png)

На данной вкладке есть возможность создать заявку на потребность закупки материалов. Для этого нужно выбрать материал с видом **покупной** и нажать **Создать заявку на потребность**. 

![](image-11.png)

Созданная таким образом заявка будет отражена в **Связках** плана производства. 

![](image-12.png)

По кнопке **"Показать узлы"** можно посмотреть узлы с видом "ДСЕ" и так же увидеть разузловку по данным дсе.

![](image-13.png)

Если у узла есть карта качества, она отобразится в соседней вкладке. В ней можно перейти к карте качества или в заявку от клиента, кликнув по интересующему вас документу.

![](image-14.png)

По кнопке **"Остатки"** в всплывающем окне выбираем склад, чтобы узнать какие номенклатуры и в каком количестве они есть на этом складе, в табличной части заполнится колонка "Остатки на складе".

![](image-16.png)

По кнопке **"Схема процессов"** откроется документ "Схема" в отдельной вкладке. В этом документе можно посмотреть весь путь производства от заявки от клиента до отгрузки и провалиться в соответствующие документы.

![](image-15.png)

На вкладке **План** по кнопке **"Назначить ответственных"** откроется окно где из выпадающего списка можно выбрать оборудование и сотрудника, либо выбрать рабочий центр и информация об оборудовании и сотруднике заполнится автоматически, затем нажать "Назначить исполнителей" в нижней части экрана.

![](image-19.png)

По кнопке **"Создать карты качества"** доступно создание карты качества по изделию. Для это нужно выделить номенклатуру и нажать **Создать карты качества**. 

![](image-17.png)

!!! Примечание
    Важно, что для создания карты качества по данному изделию обязательно наличие технологической карты. 

Карты качества создаются на все номенклатуры, которые имеют вид "ДСЕ", их можно увидеть в разузловке.

![](image-18.png)

Во вкладке **"Расчет баланса времени"** можно увидеть информацию о времени, которое понадобится для изготовления определенного узла. Для этого нужно нажать на соответствующую кнопку.

![](image-20.png)

По кнопке **"Операции"** в верхнем меню, можно загрузить неисполненные заявки с предыдущей версии плана.

![](image-21.png)

Для сохранения изменений, внесенных в документ, можно использовать кнопки сохранения. Если изменений не было или нет необходимости их сохранять использовать кнопку **Закрыть**. 

![](image-22.png)
