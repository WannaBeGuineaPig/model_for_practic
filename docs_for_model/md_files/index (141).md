# Обзор заработных плат и компенсаций

## Описание

**Обзор заработных плат и компенсаций** — документ, содержащий информацию об исследовании уровней заработных плат и компенсаций для определенных должностей. При проведении кадровых перемещений данные обзора используются для определения руководством , насколько конкурентоспособны заработные платы сотрудников и принять меры по их корректировке, если это необходимо.

Документ в Системе позволяет фиксировать ежегодные итоги проведенного анализа заработных плат и компенсаций консалтинговым агентством (контрагентом). Данные из документов применяются при составлении служебных записок при приеме нового сотрудника и позволяют провести сопоставление назначенной зарплаты сотрудника на соответствие рыночным стандартам.


## Работа с формой

Для перехода в документ необходимо в меню в поисковой строке ввести **«Обзор заработных плат и компенсаций»** и в открывшемся реестре нажать кнопку **"Создать"**. Форма доступа пользователю с определенными правами. Для доступа обратитесь к администратору системы.

В открывшейся форме необходимо заполнить поля:

* «Контрагент» - наименование консалтингового агентства, предоставившего обзор заработных плат и компенсаций, 
* «Дата начала действия с» - дата начала действия каждого следующего обзора.

![alt text](media/image-7.png)

Чтобы добавить новую позицию, необходимо нажать на + в табличной части. В созданной строке необходимо заполнить информацию: 

* **ID позиции** - шифр должности (текстовое поле);
* **Код обзора ЗП** – шифр должности консалтингового агентства, (текстовое поле);
* **Должность из обзора ЗП** – наименование должности консалтингового агентства, (текстовое поле);
* **Ежемесячная ЗП по обзору, 3-ий квартиль** – показатель обзора рынка труда, который ориентирует работодателя на установление конкурентоспособной заработной платы для привлечения ценных работников (суммовое поле).

![alt text](media/image.png)

В системе есть возможность загрузки документа с заполненными данными, которые автоматически будут добавлены в систему. Для этого необходимо в панели инструментов нажать на кнопку «Работа с шаблонами» и выбрать «Скачать шаблон».

![Alt text](media/image-3.png)

Автоматически будет скачан документ в формате xlsx, который необходимо будет заполнить и сохранить файл.

![alt text](media/image-1.png)

Для загрузки документа в систему необходимо нажать «Работа с шаблонами» и выбрать «Загрузить из шаблона».

![Alt text](media/image-5.png)

После загрузки документа через некоторое время все данные будут автоматически добавлены в таблицу.

![alt text](media/image-2.png)

Для сохранения документа нужно нажать на кнопку «Сохранить».
