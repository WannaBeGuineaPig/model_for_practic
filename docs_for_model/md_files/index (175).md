# Актуальный график отпусков

## Описание

В данной форме отображается актуальный график отпусков сотрудников предприятия. Он строится на основе [графика отпусков](../../Руководство для HR/График отпусков/) и заявлений на перенос ежегодного оплачиваемого отпуска.

## Работа с формой

Для работой с формой необходимо перейти в **Меню - Кадровый документооборот.Реестры - Актуальный график отпусков**. Или через иконку "**Меню**" в поисковой строке ввести "**Актуальный график отпусков**".  При открытии автоматически сформируется табличная часть с периодами отпуска. По умолчанию отображается текущая организация, текущий год и текущий сотрудник.

В **зеленых ячейках** отображается периоды отпуска согласно графика отпусков либо перенесенные даты согласно заявления, которые завершены по маршруту согласования.

В **красных ячейках** отображается заявления на перенос отпуска, у которых маршрут не завершён.

В **серых ячейках** подкрашиваются дни, которые отсутствуют в текущем месяце.
