# План по отгрузке

Эта аналитическая форма представляет собой плановый документ по отгрузке продукции на определенный период. Для открытия реестра документов необходимо зайти в **Меню** – Найти модуль **CRM** – Открыть форму **План по отгрузке**.

![](1.png)

Откроется аналитическая форма, в которой можно выбрать период, за который требуется увидеть план по отгрузкам:

![](2.png)

Выбираем период, например, весь год и нажимаем “Выбрать”:

![](3.png)

При необходимости выберите Группу номенклатуры. Далее нажмите на “Сформировать”:

![](4.png)

Отобразится таблица со следующей информацией: менеджер, контрагент, регион, номенклатура, ЕИ, планируемые и фактические результаты, итоги.

Столбцы **"%"** высчитывают по формулам:

- % выполнения после столбца **"Факт"** рассчитывается по формуле: кол-во факт/кол-во план * 100
- % выполнения после столбца **"Итого"** рассчитывается по формуле: кол-во итого/кол-во план * 100

При нажатии на менеджера откроется расшифровка. Отчет по отгрузке формируется по таким документам как Расходная накладная (факт), Заявка от клиента (план), План продаж.

![](5.png)

При нажатии на наименование документа он откроется.

![](6.png)
