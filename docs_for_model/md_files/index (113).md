# Реестр документов обмена с 1С

Аналитическая форма предназначена для просмотра реестра обмена документов с 1С.

![Рисунок 1 РеестрОбмена1](media/reestrObmena.png)

Для открытия формы необходимо в меню "Обмен данными с 1С" выбрать пункт "Реестр документов обмена с 1С".

Выборка производится по параметрам:

- Организация;
- Работник, физическое лицо;
- Документ;

Загрузится список документов с основной информацией по документам, прошедшим обмен с 1С.
В колонке "Код" содержатся ссылки, по которым можно открыть соответствующий коду документ.

В табличной части формы представлена информация о первичном документе, маршруте документа, информации о подписании, о данных документа из 1С, данные Документа на подписание.

Для пользователя обладающим специальными правами настроена возможность удаления связи между заявлением/документом на подписанием и системой 1С.
