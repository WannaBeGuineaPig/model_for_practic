# Заявления от сотрудников

## Общее описание назначения документа

Документ **"Заявления от сотрудников"** предназначен для создания документов-заявлений, которые требуется отправить по маршруту к определенным сотрудникам для утверждения. Например, к данной группе относятся: Заявление на отпуск для всех, Заявление на дополнительно оплачиваемый день отдыха за сдачу крови, Заявление на дополнительно оплачиваемый день, Заявления на отпуск без сохранения заработной платы и т.д.

Рассмотрим процесс оформления заявления на отпуск в системе электронного документооборота.

![type:video](https://www.youtube.com/embed/HsNIt2ZWmsI)

Чтобы создать новое заявление необходимо в панели виджетов нажать кнопку "Создать заявление"

![Виджет](media/vidjet.png)

Либо в плиточном меню выбрать пункт "Создать заявление"

![Плиточное меню](media/tiledMenu.png)

## Работа с документом

В открывшейся карточке в поле "Подтип" необходимо выбрать нужный вид заявления.

![Рисунок 5 Документ "Заявления от сотрудников"](media/image5.png)

![Рисунок 6 Результат выбора шаблона документа](media/screen_1.png)

Так же в данной форме присутствует галочка "Конфидециально". Она позволяет скрывать документы от всех пользователей кроме тех кто указан в маршруте и имеет настройку в форме "Подтипы документов"

![Рисунок](media/konfid2.png)

![Рисунок](media/konfid.png)

После нажатия автоматически заполнится бланк заявления, расположенный справа:

![Рисунок 7 Пример заполнения бланка заявления](media/screen_2.png)

Если для данного документа настроено автоматическое формирование маршрута, то появится информационное сообщение с предложением отправить документ далее по маршруту:

![Конфирм](media/confirm.png)

    Текст всплывающего сообщения можно отредактировать в настройках учетной политики. Для этого обратитесь к администратору сайта.

При согласии заявление будет автоматически направлено по маршруту согласования. В случае несогласия (например, необходимо скорректировать реквизиты документы), заявление не будет направлено по маршруту. После корректировки чтобы подписать заявление, в панели инструментов нажмите кнопку "Подписать ЭП" и в открывшемся окне нажмите "ОК".

![Рисунок 10 Выбор статуса](media/image13.png)

![Рисунок 12 Изменения в маршруте после подписания ЭП](media/signed.png)

Если для данного документа не настроено автоматическое формирование маршрута, то для формирования маршрута нужно нажать Маршрут на панели инструментов.

![Рисунок 14 Маршрут согласования](media/screen_3.png)

Далее нажать кнопку "Отправить" и выбрать статус "Подписан ЭП".

![Рисунок 15 Выбор статуса](media/image10.png)

В маршруте "Статус этапа" сменяется на "Завершен", а "Статус документа" – на "Подписан ЭП".

После подписания документ закроется и осуществляется переход на главную страницу, в случае если отсутствуют другие открытые вкладки.

## Обязательное вложение

Для некоторых типов заявлений обязательным является вложение определенных документов. Например, для заявления о предоставлении стандартного налогового вычета на ребенка, необходимо вложить свидетельство о рождении. Для этого необходимо в панели инструментов выбрать кнопку "Вложения".

![Вложение](media/vlozhenie.png)

При нажатии на кнопку «Добавить вложение» откроется диалоговое окно для выбора файла. В окне нужно выбрать нужный файл и добавить его по кнопке "Открыть".

![Свидетельство](media/svidetelstvo.png)

**Обязательным является заполнение поля Тип вложения:**

![Рисунок 9 Тип вложения](media/ТипВложения.png)

Если этот файл не включен во вложение, система оповестит об этом при отправке по маршруту. Текст предупреждающего сообщения будет следующий: _"Необходимо вложение с типом "Копия свидетельства о рождении"_.

![alt text](media/image.png)

Подробнее о работе со вложениями можно ознакомиться в разделе [Работа с вложениями](../Работа со вложениями/)

## Тиражированные заявления

Если сотрудник трудоустроен на нескольких организациях холдинга и/или принят также внутренним совместителем на одной из организаций, то при оформлении заявления на отпуск, он должен согласовать заявление по каждому месту работы. Для этого реализовано автоматическое тиражирование данного документа во все организации, и все должности внутри одной организации. Документы будут копироваться на все предприятия, где по коду физического лица, будет находиться данный сотрудник, а также на все должности, если сотрудник является внутренним совместителем автоматически по завершении маршрута. В случае, если сотрудник находится только в одной организации заявления копироваться не будут.

## Реестр заявлений

Пользователям с определенными правами доступен реестр заявлений.

В столбце **"Статус подписания"** отображается на какой стадии подписания находится электронный документ.

* _"Подписан"_ с зеленым значком - документ полностью завершен по маршруту согласования;
* _"На подписании"_ с желтым значком - документ находится на стадии согласования;
* _"На подписании!"_ с голубым значком - документ находится на стадии согласования, но в маршруте присутствует отказ от подписания электронной подписи одним из участников маршрута и был отправлен далее по маршруту;
* _"Отклонен"_ с красным значком - документ не был согласован по маршруту согласования **электронной подписью** и был возвращен инициатору;
* _"Не согласован"_ с коричневым значком - документ не был согласован по маршруту согласования;
* _"Запрос аннулирования"_ - по документу был составлен акт аннулирования и отправлен по маршруту согласования;
* _"Аннулирован"_ - документ аннулирован.

В столбце **"Подписан КЭП"** отображается зеленый значок "Подписан КЭП", в случае наличия как минимум одной подписи КЭП в завершенном этапе маршрута. Для настройки обратитесь к администратору системы.

## Создание корректирующего заявления

Работникам, работающим по трудовому договору по совместительству, оплачиваемые ежегодные трудовые отпуска предоставляются одновременно с отпуском по основной работе. Если продолжительность оплачиваемого ежегодного трудового отпуска по трудовому договору по совместительству меньше продолжительности отпуска по основной работе, работодатель по просьбе работника-совместителя предоставляет ему отпуск без сохранения заработной платы на дни, составляющие разницу в продолжительности отпусков. Соответственно, сотрудник уходит в отпуск одновременно в двух компаниях. Для этого на проекте присутствует возможность подачи корректирующего заявления работником-совместителем. Пользователь должен обладать специальными правами.

В личном кабинете [сотрудника](../../Руководство линейного сотрудника/Профиль сотрудника/) во вкладке ["Мои заявления"](../../Руководство линейного сотрудника/Профиль сотрудника/#мои_заявления) необходимо открыть заявление по которому необходимо создать корректирующее заявление.

![Кабинет сотрудника](media/cabSotrudnik.png)

В панели инструментов нажать на кнопку "Операции" и выбрать "Создать корректирующее заявление".

![Корректирующее заявление](media/correct.png)

Откроется новая форма заявления. Про дальнейшую работу с документом можно ознакомиться в разделе [выше](#работа_с_документом).

## Удаление заявления

Обычный пользователь кадрового электронного документооборота имеет право перевести документ в статус ["Удалено"](../Удаление документов КЭДО/index.md). Чтобы удалить заявление, в панели инструментов нажмите кнопку "Удалить".

![Удалено](media/delete.png)
