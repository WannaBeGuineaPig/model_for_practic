# Расходы будущих периодов

## Карточка расходов будущих периодов

Расходы будущих периодов (РБП) — это затраты, произведенные организацией в предшествующем и/или отчетном периодах, но подлежащие включению в себестоимость продукции (работ, услуг) в последующие периоды деятельности организации. Без РБП невозможно получить доходы в будущем. При этом отметим, что РБП существуют только в бухгалтерском учете.

Положениями по бухгалтерскому учету предусмотрены всего два вида затрат, которые следует признавать в составе расходов будущих периодов:

- затраты, понесенные в связи с предстоящими строительными работами. Например, материалы, переданные на строительную площадку;

л- ицензионное программное обеспечение (ПО).

Существует 2 способа создания Карточки РБП: 

- По операции из Прихода услуг;

- По кнопке “Создать” из реестра РБП.

Рассмотрим способы более подробно. 

***1. Создание РПБ по операции из Прихода услуг***. 

Поступление РБП оформляется документом Приход услуг. 
Создать новый документ Приход услуг, заполнить данные по документу, добавить строки, в колонку Наименование услуги*. Колонка Код и краткое наименование автоматически заполнятся после сохранения карточки РБП. Для того ,чтобы создать новую карточку РБП в заполненном документе Приход услуг, выделить строку, по которой необходимо создать карточку РБП и по кнопке **“Операции”** выбрать **Сформировать карточку РБП**.

![](1.png)

Откроется дополнительное окно - **Выбор услуг для создания документа РБП**. 

- Красным цветом выделены услуги, на которые карточки РБП не заведены. Карточка РБП создается двойным щелчком по строке.

- Зелёным цветом выделены услуги, на которые карточки РБП заведены.

![](2.png)

По двойному щелчку мыши по строке создается новый документ Карточка РБП. В нем заполняются поля данными из Прихода услуг, недостающие данные вносятся вручную.

![](3.png)

В созданной карточке РБП заполнить/проверить правильность заполнения полей:

- Организация (контрагент поставщик РБП);

- Наименование (наименование РБП);


- Счет кредит (счет учета РБП);
  
- Начало списания (дата начала списания РБП);

- Окончание списания (дата окончания списания РБП рассчитывается автоматически от даты начала списания и кол-ва месяцев/дней списания);

- Кол-во месяцев (кол-во месяцев списания);

- Расчет по дням (признак расчета списания по дням);

- Кол-во дней (если стоит признак Расчет по дням, то расчет будет производиться по дням, а не по месяцам);

- Подразделение (подразделение куда будут списываться затраты по РБП);

- Счет затрат (Счет дебет отнесения затрат при ежемесячном списании РБП);

- Статья затрат (статья списания по счету затрат);

- Принят в н/у (признак принятия РБП к налоговому учету);

- Доля списания в б/у, Доля списания в н/у (сумма ежемесячного списания (рассчитывается автоматически) = стоимость РБП/кол-во месяцев (дней));

После заполнения полей карточку необходимо сохранить и вернуться в документ Приход услуг и так же сохранить его.

![](4.png)

После принятия к учету документа Приход услуг в Карточке РБП в разделе Операции с карточкой появится запись о документе поступления.

![](5.png)

***2. Создание РПБ из реестра***

Для открытия реестра документов необходимо зайти в *Меню** – Найти группу **Расходы будущих периодов** – нажать на **Расходы будущих периодов (РБП)**.

![](6.png)

При выборе пункта меню, в случае, если пользователь первый раз обращается к реестру документов «Расходы будущих периодов (РБП)», появляется окно выбора расчетного периода.

![](7.png)

Откроется Реестр РБП за выбранный период. В реестре слева направо представлены реквизиты:

- Поле для выделения документа;
  
- Номер и дата документа;
  
- Наименование;
  
- Стоимость;
  
- Сумма списания;
  
- Счет (Дт/Кт);
  
- Подразделения;
  
- Статья затрат;
  
- Проект;
  
- Период списания(с/по/месяцев);
  
- Организация;
  
- Договор;
  
- Служебные поля: дата и автор создания и последнего изменения документа.

![](8.png)

При нажатии на кнопку «Создать» создается новый документ «Расходы будущих периодов». Некоторые поля на форме заполняются по умолчанию:

- Поле “№ карточки”;

- Поле “Дата” заполняется текущей расчетной датой;

- Поле “Счет кредит” заполняется значением из констант;

- Поле “Начало списания” берется из поля “Дата”;

- Поле “Принять в н/у” проставляется автоматически (если не нужно принимать в н/у необходимо убрать галочку).

![](9.png)

**Обязательные поля для заполнения:**

- Поле "Наименование";

- Поле “Окончание списания” рассчитывается автоматически после ввода количества месяцев;

- Поле “Счет затрат” выбирается из справочника;

- Поле “Статья затрат” выбирается из справочника;

- Поле “Подразделение” выбирается из справочника.

![](10.png)

Остальные поля формы:

- Поле “Расчет по дням” (Можно выбрать расчет по дням, поставив галочку в поле);

- Поле “Окончание списания”. Дни рассчитываются автоматически после заполнения “Кол-во месяцев”;

- Поле “Организация” выбирается из справочника;

- Поле “Договор” выбирается из справочника (фильтруется по организации);

- Поле “Проект” заполняется названием проекта из выбранного договора;

- Поля “Доля списания в б/у” и “Доля списания в н/у” рассчитываются автоматически от оборотов;

Табличная часть заполняется автоматически по оборотам на текущий выбранный период.

После того как документ заполнен, необходимо его сохранить по кнопке “Сохранить” на панели инструментов. После обновления страницы правая таблица заполнится, если имеется сальдо либо обороты. 

![](11.png)


## Списание расходов будущих периодов

***Общее назначение документа***

Данная операция вносит в состав расходов, уменьшающих базу по налогу на прибыль, расходы будущих периодов, сумма которых рассчитана следующим образом: если на момент проведения закрытия месяца на счетах учета расходов будущих периодов в бухгалтерском и налоговом учете числятся дебетовые сальдо, то происходит расчет сумм списания расходов будущих периодов, по правилам, указанным в карточке «Расходы будущих периодов».

***Реестр***

Для просмотра реестра или создания нового документа Инвентаризационной описи необходимо в Меню перейти в **Расходы будущих периодов – Списание РБП**.

![](1-1.png)

Откроется реестр Списаний РБП:

![](1-2.png)

***Создание документа***

Для создания нового документа необходимо нажать на кнопку "Создать" на верхней панели инструментов.

![](1-3.png)

Откроется новый документ:

![](4-1.png)

Необходимо заполнить поля:

* **Номер** – задается автоматически при сохранении документа;

* **Дата** – обязательное поле, заполняется конечной датой текущего месяца;

* **Статус** – по умолчанию заполнен значением “создан”. По необходимости можно поставить другое значение, выбрав его из списка предоставляемых;

* **Примечание** – поле, которое заполняется по желанию. Формируется автоматически на основании выбранных поля **Дата**.Также можно заполнить поле вручную.

По кнопке “Сформировать” (находящейся на панели управления) заполняется табличная часть.

![](5-1.png)

Если нужно удалить строку в документе, то надо встать на строку и нажать на кнопку «Удалить»

![](6-1.png)

Чтобы сохранить документ, нажмите на “сохранить” или “сохранить и закрыть” на панели управления.

![](7-1.png)

После сохранения документ проводится. Проводки по документу можно посмотреть по кнопке «Проводки» – «Текущие проводки» на панели инструментов.

![](8-1.png)

Выйдет окно с данными по проводкам.

![](9-1.png)
