# Статус заявки на закупку/потребность

Когда вы создаете заявку на закупку, ей присваивается статус. Тоже самое происходит и в заявке на потребность. 
Этот документ поможет создать статусы, которые используются на вашем предприятии.

Для работы с данным документом потребуется перейти в **Меню**  – найти группу **Управление закупками** – открыть документ **Статус заявки на закупку/потребность**. 

![](image.png)

Нажмите на **Статус заявки на закупку/потребность**, откроется реестр. 

![](image-1.png)

Чтобы открыть ранее созданный документ требуется нажать на него. 

Чтобы создать новый документ, требуется нажать на кнопку **Создать**, которая находится в левом верхнем углу реестра.

Откроется форма создания Статуса заявки на закупку, потребность

![](image-2.png)

В документе пристуствуют такие поля, как:

* **Наименование**, **Внутреннее наименование** – текстовое поле документа. Необходимое наименование прописывается вручную.

* **Значение по умолчанию**. В документах, где присутствует статус заявки, ваше наименование будет всегда устанавливаться по умолчанию.

Для того, чтобы Сохранить, Сохр.закрыть, Закрыть документ, воспользуйтесь соответствующими кнопками в шапке документа. 

![](6.jpg)

Для того, чтобы удалить документ из реестра, необходимо выделить его галочкой и удалить. 

![](image-4.png)
