# Исполнение по заявкам на потребность

Аналитическая форма “Исполнение по заявкам на потребность” предназначена для анализа данных по заявкам на потребность. Найти ее можно в гглавном меню системы.

![](image.png)

Форма выглядит следующим образом:

![](image-1.png)

После открытия аналит.формы выберите период, за который хотите загрузить Заявки на потребность. Для этого используйте соответствующую кнопку:

![](12.png)

Затем нажмите кнопку Сформировать:

![](image-2.png)

Заполнится табличная часть, в которой отображается следующая информация: 

* Какая организация создала Заявку на потребность, заявка на потребность;

* Наименование материалов, единицы измерения, возможную Замену;

* Статусы потребности - тянутся из табличной части заявки на потребность: «Создано» - если закупка из потребности не создана. Когда закупка создана, статус потребности меняется «в работе»;
  
* Статусы закупки - тянутся из заявки на закупку в табличной части;

* Проекты, дату окончания проектов;

* Информация об инициаторе, исполнителе и поставщике;

* Дату поставки (Потребность/Покупка/Спецификация/Факт/Оплата);

* Количество (Потребность/Перемещено/Закупка/Спецификация/Факт);

* Сумму без НДС (Потребность/Покупка/Спецификация/Факт/Оплата);

* Договор/счет, платежное поручение, приходный ордер;

Если требуется посмотреть исполнение по конкретной Заявке на потребность/Проекту/Номенклатуре, то заполните соответствующее поле в шапке.

Например, выберите Номер заявки из выпадающего списка. После формирования в табличной части окажутся все позиции номенклатуры из указанной заявки:

![](image-3.png)

По ссылке в табличной части есть возможность перейти в Исполнение по данной номенклатуре:

![](image-4.png)

Откроется вкладка, в которой указаны Документы, Даты поставки, Количества, а также Суммы/Цены без НДС и с НДС с этой номенклатурой. По клику на одно из наименований документов в табличной части осуществится переход в него.

![](image-5.png)

Вернитесь в **Исполнение по заявкам на потребность**. 

## Создание закупки 

Выделите одну или несколько позиций, по которым еще не создавалась Заявка на закупку. Для этого одновременно зажмите клавишу CTRL + левую кнопку мыши на нужных строках с материалами, затем нажмите на Создать закупку:

![](image-7.png)

Откроется Заявка на закупку, включающая в себя эти позиции:

![](image-6.png)

Заполните документ и сохраните его. 

## Создание Спецификации 

Выделите одну или несколько позиций, по которым еще не создавалась Спецификация. Для этого одновременно зажмите клавишу CTRL + левую кнопку мыши на нужных строках с материалами, затем нажмите на Создать спецификацию:

![](image-8.png)

Откроется окно выбора Контрагента, Договора и Приложения к договору. Заполните эти поля выбором из выпадающего списка, затем нажмите Сформировать:

![](image-9.png)

Откроется созданное Приложение к договору. Во вкладке Спецификация отобразится номенклатура, на основании которой создался документ. 

![](image-10.png)

Заполните документ и сохраните его. 

После сохранения Спецификации статус в аналит.форме Исполнение по заявкам на потребность принял вид “Заключение договора”.

![](image-11.png)
