# Схема

Документ **"Схема"** нужен для наглядного управления производственными заявками, отслеживания их движения по этапам (от потребности до отгрузки) и служит центральным элементом для координации действий между различными отделами компании в рамках автоматизированного бизнес-процесса. Это инструмент оперативного планирования и контроля.

Для того, что бы открыть схему, нужно в лукапе "Бизнес-процесс" выбрать тот, по которому мы хотим отобразить схему.

![](image.png)

После этого откроется сама схема.

![](image-1.png)

Кликнув дважды по этапу, можно перейти к этому документу.

![](image-2.png)

!!! Примечание

    Если в Заявке от клиента несколько номенклатур, то в схеме они отобразятся отдельно.
