# Сдельные наряды

**Сдельные наряды** — это первичные учетные документы, используемые при сдельной системе оплаты труда, которые определяют объем работы, сроки, расценки и служат основанием для расчета зарплаты работникам.

Документ можно найти через поиск в главном меню системы в модуле "Управление производством"

![](image.png)

После нажатия на **"Сдельные наряды"** откроется реестр документов. 

![](image-1.png)

Для создания документа нужно нажать на кнопку ![](image-2.png)

Откроется форма документа, где автоматически заполнены поля:

* **Организация** - Заполняется та, в который вы находитесь. Поле не редактируемое.
* **Дата** - Заполняется текущей датой, есть возможность редактирования.
* **Статус** - По умочанию заполняется значением "Создан", меняется на "Закрыт", когда работа выполнена, а исполнителю начислена зарплата.
* **Номер** - Заполняется автоматически после сохранения документа.
* **Подразделение** - Заполнится автоматически при выборе "Исполнителя".

![](image-3.png)

Обязательные поля:

* **Исполнитель** - Выбирается вручную из выпадающего списка. (Из справочника "Сотрудники").

Прочие поля:

* **Заявка от клиента** - Если работа выполнена по заказу, то ее можно заполнить из выпадающего списка, если нет - оставить поле пустым.
* **Дата закрытия** - Заполняется вручную, после того как ставится статус "Закрыт".
* **Прмечание** - Заполняется вручную дополнительной информацией.

Для добавления строк в табличную часть, нужно нажать на ![](image-4.png).

В ней заполняются такие поля, как:

* **Дата** - Дата выполнения операции. В этот день будет начислена зарплата за заказ.
* **Заявка от клиента** - Заполнится автоматически, если в шапке есть заявка от клиента. Если в шапке поменять заявку от клиента, то появится всплывающее окно с вопросом, хотим ли поменять заявку в таблице.
* **Карта качества, Технологическая карта, Изделие, ЕИ** - Заполнятсся автоматически из Заявки от клиента.
* **Операция** - Операция, которую необходимо выполнить.
* **Количество(план)** – Запланированное количество выпуска продукции. 
* **Количество (факт)** – Фактическое количество выпуска продукции.
* **Норма** – Норма времени для изготовления единицы продукции, берется Из технологической карты.
* **Тарифная ставка** – Стоимость выполнения операции для изготовления единицы продукции.
* **Нормочасы** – количество нормочасов.
* **Стоимость выполненных операций** - Нормочасы умноженные на Тарифную ставку.
