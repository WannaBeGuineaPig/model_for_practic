# Проверка вложений для выгрузки в МинТруд

## Описание

**Проверка вложений для выгрузки в МинТруд** - аналитическая форма для анализа вложений в документах. Согласно [приказу Минтруда России от 20.09.2022 № 578н](http://publication.pravo.gov.ru/Document/View/0001202209300050?index=0&rangeSize=1) к составу и форматам документов, связанных с работой и оформляемых в электронном виде без дублирования на бумажном носителе имеются единые требования. В данной форме ответственными специалистами производится анализ вложений электронных документов согласно данных требований. **Доступ к данной форме имеет пользователь с определенными правами.**

## Работа с формой

Для открытия документа необходимо в Меню в поисковой строке ввести - Проверка вложений для выгрузки в МинТруд. Откроется форма.

![Форма](media/form.png)

В поле **"Тип документа"** выбирается тип документа.

![Тип документа](media/tip.png)

В поле **"Подтип документа"** - выбирается подтип документа. 

![Подтип документа](media/podtip.png)

В поле **"Период создания"** указывается период создания электронный документ.

![Период](media/period.png)

Для загрузки вложенных документов нужно нажать на кнопку **"Загрузить вложения из документов"**. 

![Загрузка](media/load.png)

После этого в табличной части формы появятся все вложения к электронным документам, относящимся к выбранному типу и подтипу документа, созданным в указанный период.

![Таблица](media/table.png)

В столбце **"Документ"** содержится гиперссылка на электронный документ.

![Документ](media/document.png)

В столбце **"Классификатор минтруда"** отображается код подтипа документа, соответствующий классификатору МинТруда России.

![Классификатор](media/class.png)

В столбце **"Файл"** указывается название вложения к электронному документу.

![Файл](media/file.png)

В столбце **"Скачать"** можно скачать вложение при необходимости. 

Столбец [**"Главный документ"**](../../СЭД/Работа со вложениями/#главный_документ) позволяет определить, какое вложение является главным документом среди вложений.

![Главный документ](media/glavDoc.png)

В столбце **"Ошибка"** отображаются ошибки в структуре документа формата XML, если таковые имеются.

![Xml](media/xml.png)

Если в документе формата XML имеются ошибки, то его можно заменить на новый. Для этого необходимо выбрать вложение и нажать на кнопку **"Заменить файл XML"**.

![Новый xml](media/newXML.png)

Появится информационное сообщение о создании нового файла xml.

![Уведомление](media/uvedom.png)

Чтобы новый сгенерированный файл xml автоматически будет заменен во вложениях к документу.

![Новый xml](media/newXML2.png)
