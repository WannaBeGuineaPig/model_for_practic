# Типы дополнительного документа к договору

## Общее описание назначения документа

**”Типы дополнительного документа к договору"** предназначены для составления списка возможных видов дополнительного документа к договору.

## Реестр

Для просмотра реестра "Виды дополнительного документа к договору" и/или создания нового документа, необходимо в Меню перейти в **Учет договоров – Учет договоров. Справочники– Типы дополнительного документа к договору.**

![](2.jpg)

Откроется реестр документов:

![](1.png)

## Cоздание документа

Для создания нового документа необходимо нажать на следующую кнопку: ![](3.jpg)

Откроется новый документ: 

![](3-1.jpg)

* **Наименование** – поле, которое обязательно для заполнения. Данные вносятся вручную. 

* **Родительный падеж, Примечание** – поле, данные которого вводятся вручную. Не является обязательным для заполнения. 

* **Действует с" и "по"** можно указать период времени, в течение которого данный тип дополнительного документа будет действовать в системе. По истечению заданного срока при создании документов данный тип не будет предложен. Если даты в полях не указаны, то тип документа считается бессрочным.

После заполнения документа сохраните его.
