# Очистка кэша

У пользователя после обновления версии программы могут возникать сообщения об исключительных ситуациях в работе программы.

![Рисунок 1 Аппаратная перезагрузка и очистка кэша](./media/image1.png)

Перед обращением в техподдержку необходимо очистить закэшированные данные, которые могут провоцировать некорректную работу программы.

Для этого на своем рабочем месте произвести перезагрузку с очисткой кэша. Для этого необходимо открыть интернет-браузер.

## Браузер Google Chrome

Если интернет-браузер **Chrome**, очистка кэша происходит следующим образом:

1. Нажать клавишу F12 (открыть консоль разработчика, в зависимости от настроек компьютера возможна комбинация клавиш Fn+F12).

2. Нажать правой кнопкой мышки на стрелку обновления страницы 

![Рисунок 2 Очистка кэша и жесткая перезагрузка](./media/image2.png)

3. В выпадающем списке выбрать пункт «Очистка кэша и жесткая перезагрузка»

4. Закрыть консоль разработчика

## Браузер Microsoft Edge

Аналогично выполняется очистка кэша в браузере **Microsoft Edge**

1. Нажать F12

2. Нажать правой кнопкой мышки на стрелку обновления страницы

![Рисунок 3 Очистка кэша и принудительное обновление](./media/image3.png)

3. В выпадающем списке выбрать пункт «Очистить кэш и выполнить принудительное обновление»

4. Закрыть консоль разработчика

## Яндекс Браузер

1. Зажать комбинацию клавиш **Ctrl + Shift + Del**

2. В появившемя окне нажать на **"Очистить"**

![Alt text](media/image-1.png)

## Mozilla FireFox

1. Зажать комбинацию клавиш **Ctrl + Shift + Del**

2. В появившемя окне нажать на **"Удалить сейчас"**

![Alt text](media/image-2.png)

## Safari

1. В панели инструментов нажать Safari и выбрать очистить историю

![Alt text](media/image-3.png)

2. В появившемя окне нажать на **"Удалить сейчас"**

![Alt text](media/image-4.png)

## Браузер "Спутник"

1. В правом верхнем углу браузера нажать по кнопке с тремя точками, в появившемся меню выбрать пункт Настройки.

![Alt text](media/image-5.png)

2. В настройках выбрать пункт **Дополнительные**, далее пункт Безопасность и конфиденциальность. Далее нажать на пункт **Очистить историю**.

![Alt text](media/image-6.png)

3. Нажать на кнопку **Удалить данные**.

![Alt text](media/image-7.png)

Если после очистки кэша сообщение не исчезло, необходимо обратиться в техподдержку.
