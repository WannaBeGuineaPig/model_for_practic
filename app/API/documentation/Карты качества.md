# Карты качества

Документ Карта качества служит для систематизации и управления процессом производства и контроля качества конкретного изделия. 

Есть 2 способа создания Карты качества:

1. Из документа **План производства** (только если есть тех.карта)
2. Из реестра Карт качества по кнопке **Создать**

## Создание документа из Плана производства

Для того, чтобы создать Карту качества из Плана производства необходимо во втором документе выделить номенклатуру галочкой, затем нажать кнопку **Создать карты качества**, предварительно разузловав документ по соответствующей кнопке.

В правом углу документа появится сообщение об успешном создании документа. Карты качества создаются на все позиции, вид которых "ДСЕ", их можно посмотреть во вкладке разузлование.

![](image.png)

!!! Примечание 
    Карта качества создастся только в том случае, если в Плане производства указан Состав изделия и по этому составу есть Тех.карта!

По кнопке **Связки** можно перейти в созданные Карты качества:

## Создание документа из реестра по кнопке Создать

Для работы с документом откройте реестр **Карты качества**, нажмите кнопку **Создать**:

![](image-2.png)

Откроется документ, который состоит из шапки документа и табличной части.

![](image-3.png)

В шапке документа нужно заполнить такие поля как:

* **Изделие** – выбирается из выпадающего списка или Обозревателя аналитик. По необходимости осуществляется переход в выбранное изделие
* **Состав изделия** – Перечисляет компоненты и материалы, используемые в изделии. Выбирается из выпадающего списка после выбора Изделия
* **Тех.карта** – Описывает этапы производства и технологические процессы, необходимые для изготовления изделия. Выбирается из выпадающего списка после выбора Изделия
* **Плановое кол-во** – указывается запланированное количество изделий для производства
* **Ед.изм.** – выбирается из выпадающего списка
* **Статус** – по умолчанию заполняется значением Создано. По необходимости меняется на другое значение, которое выбирается из выпадающего списка
* **Заявка от клиента** – выбирается из выпадающего списка. По необходимости осуществляется переход в выбранную заявку от клиента
* **Покупатель** – указывается автоматически из выбранной Заявки от клиента. По необходимости осуществляется переход в указанного покупателя
* **Склад комплектующих** – место, где хранятся компоненты изделия. Выбирается из выпадающего списка
* **Склад производства** – место, где осуществляется сборка и обработка изделий. Выбирается из выпадающего списка
* **Склад ГП** – место, где хранятся готовые изделия перед отправкой. Выбирается из выпадающего списка
* **Проект** – выбирается из выпадающего списка
* **Номер, Дата** – заполняются автоматически при сохранении документа

![](image-4.png)

В табличной части заполняются такие данные как:

* **Операция** - определенный этап производственного процесса, который может подлежать контролю для подтверждения соответствия установленным нормам. Выбирается из выпадающего списка.
* **Требуется ОТК** - чекбокс требующий пройти **Отдел технического контроля**.
* **Отметка ОТК** - чекбокс, который заполняется автоматически после прохождения контроля в документе **"Исполнение ОТК"**.
* **Контролер** - заполняется автоматически при отметке ОТК в документе **"Исполнение ОТК"**.
* **Дата ОТК** - заполняется автоматически при отметке ОТК в документе **"Исполнение ОТК"**
* **Выполнено** - заполняется, когда в документе **"Исполнение производственных операций"** все операции выполнены.
* **Завершено/Изолятор** - завершенное количество заполняется автоматически, , когда в документе **"Исполнение производственных операций"** все операции выполнены. Изолятор заполняется вручную, если есть какой то брак, при этом в количестве "Завершено" отнимется количество брака.

![](image-5.png)

* **Дата начала/Дата окончания** - заполняется исходя из документа **"Исполнение производственных операций"**.
* **Исполнение** - при нажатии открывается окно с сотрудниками, их должностями, разрядами и временем, которое они затратят на обработку данного изделия.

![](image-6.png)

* **Оборудование** - при нажатии открывается окно с оборудованием, нужным для данного этапа.

![](image-7.png)

Сохраните документ по соответствующей кнопке. 

![](image-8.png)

В верхнем меню по кнопке **"Необходимые материалы"** можно создать акт списания.

![](image-10.png)

По кнопке **Проводки** отобразится Журнал проводок. 

![](image-9.png)
