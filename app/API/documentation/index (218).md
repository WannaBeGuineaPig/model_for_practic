# Критерий оценки теста

С помощью критериев можно обеспечить надежность и независимость оценивания

Критерий оценивания — это средство, которое можно использовать для оценки работ. Создавая критерий оценивания, вы делите назначенную работу на отдельные части.

## Создание критерий

Для открытия документа необходимо в Меню в поисковой строке ввести - "**Критерий оценки теста**". У пользователя должна быть предоставлена роль **StaffTraining_Admin**.

В открывшемся документе нажмите кнопку "Создать".

В поле "Наименование" введите название критерий оценки. Например "Сдал / Не сдал"

В табличной части добавьте название оценки и соотношение правильных / неправильных ответов. Например, не сдал в случае правильности ответов от 0% до 79%, а сдал от 80% до 100%. Нажмите Сохранить и Закрыть.
