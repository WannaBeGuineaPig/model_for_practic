# Производственный отчет

**"Производственный отчёт"** — это аналитическая форма, содержащая информацию о ключевых показателях производственного процесса для оценки эффективности и оптимизации.

![](image.png)

Для того, что бы построить отчет нужно выбрать интересующий вас период и склад (есть возможность выбрать несколько)

![](image-1.png)

Так же есть возможность построить отчет фильтруя его по Продукции и Проекту. 

Нажмите на кнопку **"Сформировать"**, отчет построится по выбранными вами параметрами.

![](image-2.png)

Из построенного отчета можно перейти в Документ или в Источник

![](image-3.png)

По кнопке ![](image-4.png) можно скрыть левое меню с фильтрами.

![](image-5.png)
