# Периодичность обучения

В данном справочнике указывается информация по вариантам периодичности прохождения обучения. У пользователя должна быть роль **StaffTraining_Admin**

Для создания новой периодичности,необходимо в Меню в поисковой строке ввести - "**Периодичность обучения**

В открывшемся реестре нажмите кнопку "Создать". Заполните поля "Наименование" и "Периодичность в месяцах". Нажмите "Сохранить и закрыть"
