#Виды производственных операций

Для работы с документом **Виды производственных операций** необходимо перейти в **Меню** – найти группу **Управление производством** – открыть документ **Виды производственных операций**.

![](image.png)

Нажмите на **Виды производственных операций**, откроется реестр. Чтобы открыть ранее созданный документ требуется нажать на него. Чтобы создать новый документ, требуется нажать на кнопку **Создать**

![](image-1.png)

Откроется новый документ:

![](6.jpg)

В документе нужно заполнить такие поля, как:

* **Наименование** – обязательное текстовое поле документа, которое прописывается вручную.

* **Описание** – текстовое поле, информация прописывается вручную. По желанию можно оставить пустым. 

Для того, чтобы **Сохранить**, **Сохр.закрыть** или **Закрыть** документ, необходимо нажать на кнопки, которые находятся в шапке документа:

![](8.jpg)

Чтобы удалить документ из реестра, необходимо выделить галочкой необходимый документ и нажать на кнопку "Удалить".

![](9.jpg)
