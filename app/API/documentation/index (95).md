# Удаление документов кадрового электронного документооборота

Бывают случаи когда пользователь создал электронный документ, например, заявление на ежегодный отпуск, и направил его по маршруту согласования. Но через некоторое время изменяет свое решение и передумывает, в результате чего принимает решение удалить сформированный ранее документ.

Для этого необходимо перейти в документ, и в панели инструментов нажать на кнопку "Удалить".

![Перевод документа в статус "Удалено"](media/image1.png)

**Пользователь не сможет удалить документ документ, если:**

* Документ имеет завершенный маршрут;
* Маршрут у другого участника на подписании;

**Пользователь сможет удалить документ, если:**

* У документа отсутствует маршрут;
* У документа присутствует маршрут, но документ находится у инициатора;
* Документ отозван инициатором и находится у него на этапе
