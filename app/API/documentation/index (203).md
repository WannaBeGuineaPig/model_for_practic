# Действие обученности при смене работодателя

## Создание нового действия

Для открытия документа необходимо в Меню в поисковой строке ввести - "**Действие обученности при смене работодателя**. 
У пользователя должна быть предоставлена роль **StaffTraining_Admin**.

В открывшемся реестре нажмите кнопку "Создать"

Заполните "Наименование". Нажмите "Сохранить и закрыть"
