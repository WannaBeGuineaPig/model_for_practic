# Роли системы

## Общее описание

В данном разделе приводятся роли и описание их воздействия на интерфейс пользователя в зависимости от его должностных обязанностей. При назначении ролей следует помнить о том, что для доступа к просмотру документов КЭДО других сотрудников необходима настройка [горизонтальных прав пользователя](../Настройка пользователей/#документооборот).
