# Доверенность

Для открытия документа необходимо в Меню в поисковой строке ввести - Доверенности 

Откроется реестр документов:

![Рисунок 2 Реестр документов "Доверенности на конструкторе](./media/image2.png)

В колонке **"Подписан ЭП"** отображается зелёный значок, указывающий на наличие подписи КЭП на завершенном этапе маршрута.

С помощью кнопки **"Загрузить из Контура"** в реестре загружается [МЧД](../Машиночитаемая доверенность/) из системы Контур.Доверенности.

Для создания нового документа нажмите **"Создать"**. Откроется форма для заполнения:

![alt text](media/image-7.png)

Для начала работы с документом необходимо выбрать из справочника Подтип документа.

![Alt text](media/image-1.png)

В поле **"ЮЛ/ИП/ФЛ"** выбирается категория представителя электронной доверенности: юридическое лицо, индивидуальный предприниматель или физическое лицо, филиал (обособленное подразделение юридического лица) или филиал (аккредитованное представительство) иностранного юридического лица.

## Доверитель и доверенные лица

В данном блоке заполняется информация о доверителе и доверенном лице.

В поле **Лицо без доверенности (доверитель)** указывается Ф.И.О. и должность доверителя электронной доверенности владельца квалифицированной электронной подписи. Доверитель выбирается из выпадающего списка [физических лиц](../Физические лица/index.md) либо создается новое физическое лицо. Строка **"Должность"** заполняется автоматически при выборе физического лица (если он является сотрудником в организации) из выпадающего списка, либо задается вручную.

В форме имеется возможноть перейти в карточку сотрудника указанного в таблице из выпадающего списка. Данная функция доступна по определенным правам, для этого необходимо обратиться к админитратору. 

![Alt text](media/screen1.png)

В разделе **Сведения об уполномоченных представителях** заполняются данные, в зависимости от выбранной категории в поле **"ЮЛ/ИП/ФЛ"**:

* **Физическое лицо** - представитель выбирается из выпадающего списка [физических лиц](../Физические лица/index.md) либо создается новое физическое лицо и заполняются все необходимые данные;

* **Юридическое лицо** - представитель выбирается из выпадающего списка [контрагентов](../Контрагенты/index.md) либо выбирается из обозревателя аналитик и аналогично заполняются все необходимые данные;

* **Индивидуальный предприниматель** - данные по представителю заполняются вручную;

## Сведения о полномочиях

В данном блоке заполняется информация о назначенных полномочиях для уполномоченного представителя.

**Утрата полномочий** - выбирается из выпадающего списка в зависимости от того, сохраняются ли полномочия после истечения доверенности. Заполняется при условии, что доверенность предполагает возможность **передоверия**.

**Совместные полномочия** - выбирается из выпадающего списка в зависимости от того, будут ли полномочия предоставлены только одному лицу (индивидуальные) или нескольким лицам совместно (совместные).

В табличной части **"Полномочия"** выбираются полномочия для представителя из выпадающего списка. Добавляются по кнопке **"Изменить"**.  
Откроется окно полномочий, где вверху окна находиться табличная часть. По кнопке "Добавить" добавляются полномочия для представителя.  
Так же ниже табличной части находится поле, где можно описать полномочия в свободной форме.

В табличной части **"Ограничения полномочий"** выбираются ограничивающие полномочия для представителя из выпадающего списка. Добавляются по кнопке **"Добавить"**.

## Сведения о доверенности

В данном блоке заполняется общая информация о доверенности.

**Вид доверенности** и **Без отзыва** - выбираются из выпадающего списка. **Доверенность с возможностью отзыва** означает, что доверитель имеет право аннулировать (отозвать) данную доверенность в любое время. **Без возможности отзыва** подразумевает, что доверенность не может быть аннулирована после ее выдачи и она действительна до истечения срока или достижения цели, указанной в ней.

**Передоверие** - выбирается из выпадающего списка. **Передоверие** предполагает, что уполномоченное лицо (получатель доверенности) может передать свои полномочия другому лицу. **"Без права передоверия"** означает, что получатель доверенности не имеет права передавать полномочия дальше. **"Однократное передоверие с правом последующего передоверия"** позволяет получателю передать полномочия только один раз и только одному другому лицу.

![Alt text](media/sved%20o%20dover.png)

Необходимо также заполнить атрибуты **Период действия "с" и "по"** доверенности.

## Наблюдатели

**Наблюдатели** необходимы для того чтобы документ можно было установить на контроль. Для этого необходимо в табличной части добавить пользователей которые будут наблюдателями. Выбранным пользователям приходит уведомление на почту и в документообороте на вкладке "Контроль" добавляется документ контроля.  
Так же участники маршрута имеют возможность, после отправки документа по маршруту, редактировать поле "Наблюдатели". 

![Рисунок](media/Observers.png)

## Работа с документом

После заполнения всех необходимых сведений необходимо нажать на кнопку "Заполнить шаблон". 

Далее необходимо отправить документ по маршруту, нажав на кнопку "Маршрут" на панели инструментов. Более подробно работа с маршрута описана в разделе [Редактирование маршрута](../../Работа с маршрутами/).

Можно настроить направление уведомления-напоминания инициатору и поверенным об окончании срока действия доверенности за указанное количество дней до истечения с предложением сделать заявку на выдачу новой доверенности. Для этого необходимо обратиться к администратору сайта.
