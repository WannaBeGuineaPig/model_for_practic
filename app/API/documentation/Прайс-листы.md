# Прайс-листы

## Реестр

Для открытия реестра документов необходимо зайти в **Меню** – Открыть документы **Прайс-листы**.

![](1.png)

Откроется реестр Прайс-листов:

![](2.png)

При желании можно копировать документ. Для этого отметьте документ галочкой и нажмите на кнопку "скопировать", расположенную на панели управления:

![](11(333333333).png)

## Создание документа 

Чтобы создать Прайс-лист, нажмите на соответствующую кнопку в реестре документов:

![](3.png)

Откроется документ. Перейдите во вкладку “Формулы”, чтобы настроить видимость колонок. 

- **Отображение альтернативных ЕИ** - по умолчанию галочка стоит – колонка отображается.

- **Видимость колонок** - по умолчанию заполнено значением “Только цена без НДС”. 

![](4.png)

По необходимости “Видимость колонок” примет другое значение из списка:

![](5.png)

Формулы по умолчанию пустые. Добавить строки для формул по кнопке “Добавить”

![](6.png)

В первой строке всегда выбирать Себестоимость т.к. от нее считается Пороговая и Отпускная цены. [1] - заполнится при выборе себестоимости – это присвоен номер строки. Значение участвует в формулах.

![](7.png)

Далее добавляем строки и записываем формулы в [] записываются номер строки от которой идет расчет. 

!!! note "Примечание"
     Формулы пишутся через точку(не запятую) 1.2 и 1.3 и т.д.

![](8.png)

Переходим на основную закладку, добавляем номенклатуры, которые будем считать по этим формулам, проставляем Себестоимость (в данном случае без НДС) нажимаем Рассчитать. 

![](9.png)

Чтобы распечатать документ, нажмите на соответствующую кнопку, расположенную на панели управления:

![](10.png)
