# Протокол совещания

## Описание

В данной статье описан порядок работы по оформлению протокола совещания, выдаче поручений в соответствии с принятыми решениями и контролю выполнения решений.

## Создание нового совещания

Создать протокол совещания можно двумя способами: через [форму](#работ_с_формой) либо через [виджет](#работа_с_виджетом)

### Работа с формой

Для открытия документа необходимо в Меню в поисковой строке ввести - Протокол совещания.  И в открывшемся реестре нажать кнопку "Создать". Откроется новая форма протокола совещания.

![Новый протокол](media/6.png)

В блоке **"Регистрационные данные протокола"**, заполняются номер, дата, составитель протокола, и повестка совещания. В табличной части **"Участники"** заполняются сотрудники, которые будут присутствовать на совещании. В табличной части  При необходимости возможно добавить дополнительные файлы к протоколу совещания.

После совещания, для каждого пункта повестки назначаются ответственные исполнители, которые назначаются в табличной части ниже. Для добавления нового пункта необходимо нажать на кнопку + и  заполнить следующую информацию: пункты протокола, исполнители и срок исполнения. Исполнителей можно назначить, нажав на иконку редактирования:

![Карандаш](media/5.png)

В открывшемся окне нужно выбрать одного или нескольких сотрудников из предложенного списка и нажать "Добавить". Добавленные сотрудники будут отображены как исполнители. Выбранные сотрудники отобразятся как исполнители.  После завершения внесения всех пунктов, протокол отправляется по [маршруту согласования](../Обработка документов/index.md). 

После согласования и утверждения протокола назначается [поручение](../../СЭД/Поручение/). Для этого в панели инструментов необходимо нажать на кнопку "Операции" и выбрать "Создать поручения".

![ОРД](media/4.png)

Система автоматически создаст поручения по каждому пункту, которые можно просмотреть, кликнув на ссылку в колонке "Поручение".

![Ссылка](media/screen-4.png)

![ОРД](media/screen-5.png)

Если необходимо скопировать протокол, необходимо выбрать необходимую в реестре и нажать кнопку "Копировать". Система автоматически откроет скопированный протокол, который при необходимости возможно отредактировать, и после этого запустить вновь по маршруту согласования. При копировании протокола переносятся регистрационные данные. **Созданные ранее поручения и маршрут согласования не копируются**.  Скопировать можно как с завершенным, так и не завершенным маршрутом согласования. Скопировать протокол возможно также скопировать внутри самого протокола (через кнопку "Операции"). При копировании также переносятся вложения к протоколу и связки из основного протокола при их наличии.

### Работа с виджетом

На главной странице в левой панели необходимо нажать на иконку календаря:

![Виджет](media/screen-6.png)

Для создания протокола необходимо выбрать совещание и нажать кнопку по созданию протокола:

![Протокол](media/protocol.png)

Откроется форма протокола, где автоматически будет заполнена повестка совещания и сотрудники которые участвовали в совещании.

Заполняется форма аналогично [выше описано](#работа_с_формой)
