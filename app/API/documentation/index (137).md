# Общие принципы работы с мобильным приложением

## Главная страница

После прохождения авторизации откроется главная страница, на которой необходимо выбрать - какое действие необходимо совершить. При нажатии на ["Создать новый документ"](../Создание заявления/index.md) откроется форма, в которой необходимо выбрать какой тип документа необходимо создать. При нажатии на ["Обработать документы"](../Согласование документов в мобилке/index.md) откроется реестр документов на обработку. При нажатии на ["На моем контроле"](#документы_на_контроле) откроеется реестр документов находящиеся на контроле. При нажатии на ["Личный кабинет"](#личный_кабинет) откроется форма личного кабинета сотрудника.

![Alt text](media/image.png)

## Просмотр документов

При нажатии на "Обработать документы" откроется реестр документов на обработку

![Окно с документами](media/image1.png)

В данной вкладке отображаются все документы пользователя, которые необходимо обработать. 

Подробнее как подписать и ознакомиться с документом можно прочитать в разделе [Согласование документов в мобильном приложении](../Согласование документов в мобилке/index.md), [Обработка документа в вебверсии](../Обработка документа в вебверсии/index.md).

В реестре документов выбирается документ который необходимо обработать.

После нажатия на документ открывается окно документа.

![Окно документа](media/docWindow.png)

### Вкладка "Вложения"

На данной вкладке отображаются вложенные файлы к данному документу. По нажатию на файл, можно просмотреть содержимое файла.

!!! warning
    Для возможности просмотра любого файла в смартфоне на базе Android должна быть установлена программа, функционал которой позволяет работать со многими форматами текстовых файлов, например WPS Office. В большинстве современных смартфонов такая программа установлена вместе с заводскими настройками.
    При открытии файла появится дополнительное окно, в котором смартфон предложит различные варианты работы с данным файлом.

    ![Открыть с помощью](media/openWith.png)

    В смартфонах на базе IOs текстовые файлы отображаются внутри приложения без дополнительного программного обеспечения

    ![Заявление на айфоне](media/iosZayavlenie.png)

При необходимости в документ возможно добавить необходимое вложение. Для этого необходимо нажать на кнопку **"Прикрепить документ"**.

![Прикрепить документ](media/docInit.png)

Во всплывающем окне необходимо выбрать: **"Тип вложения"** из выпадающего списка и выбрать способ добавления вложения через кнопку **"Добавить вложение"** (сделать фотографию, прикрепить имеющуюся фотографию или добавить документ). После добавления вложения, вложение будет отображаться в списке. Его можно предсмотреть, нажав по названию типа, либо удалить через иконку "корзины".

![Вложение](media/vlozhenie.png)

После этого необходимо нажать кнопку **"Прикрепить вложение"**. Вложение будет добавлено в документ.

![Прикрепить вложение](media/prikrep.png)
![Вложение](media/vlozhenie2.png)

### Вкладка "Данные"

На данной вкладке отображаются регистрационные данные документа: номер документа, работник, автор, подразделение и организация.

![Данные](media/registDannye.png)

### Вкладка "Текст"

На данной вкладке размещено основное содержание документа.

![Текст](media/text.png)

В мобильном приложении присутствует возможность просмотра текста посредством увеличения или уменьшения масштаба с помощью пальцев: для увеличения следует развести два пальца на экране, а для уменьшения - сделать обратное движение, сведя два пальца вместе.

### Вкладка "Маршрут"

На данной вкладке отображается маршрут документа

![Маршрут](media/marshrut.png)

### Вкладка "Связки"

На данной вкладке отображаются документы, которые имеют связку с текущим. При нажатии на документ произодет автоматическая переадресация на этот документ.

![Связка](media/svyazka.png)

## Документы на контроле

При нажатии на "На моем контроле" откроеется реестр документов находящиеся на [контроле](../../Начало работы/Стартовая страница/#раздел_контроль). Для просмотра документа необходимо нажать на него, после чего произодет автоматическая переадресация на этот документ.

![Контроль](media/control.png)

При отсутствии документов на контроле, мобильное приложение оповестит об этом.

![Нет документов](media/noControl.png)

## Личный кабинет

Личный кабинет сотрудника включает в себя отображение личных данных сотрудника  оставшиеся дни отпуска, отправленные документы и локальные нормативные акты.

### Персональные данные

На данной вкладке отображаются личные данные сотрудника, а также занимаемая должность.

![Alt text](media/image-1.png)

### Документы

На данной вкладке отображаются паспортные данные сотрудника, ИНН, СНИЛС, об образовании и военном билете.

![Alt text](media/image-2.png)

### График отпусков

На данной вкладке отображается график отпусков сотрудника. Чтобы развернуть график, необходимо нажать на стрелочку справа. 

![Alt text](media/image-3.png)

Также на данной вкладке возможно создать заявление на отпуск вне графика либо на перенос согласно графика.

![Alt text](media/image-4.png)

При нажатии на **"Создать заявление вне графика"** автоматически откроется [новая](../Создание заявления/index.md) форма заявления.

![Alt text](media/image-5.png)

При нажатии на **"Создать заявление на перенос"** автоматически откроется новая форма заявления, где автоматически будут заполнены атрибуты согласно графика.

![Alt text](media/image-6.png)

## Выбор компании

При нажатии на иконку профиля откроется страница, на которой будет указан сайт приложения, выбранная организация и кнопка, по которой можно выйти из системы.

Чтобы переключаться между организациями необходимо нажать на организацию и выбрать другую из списка.

![Организация](media/org.png)

Выбранная организация будет отображаться на главной странице в нижней части экрана

![Сорг](media/Corg.png)
