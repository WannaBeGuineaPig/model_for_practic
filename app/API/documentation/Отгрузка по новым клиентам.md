
# Отгрузка по новым клиентам

**"Отгрузка по новым клиентам"** — это инструмент, предназначенный для анализа данных о продажах и поставках товаров или услуг клиентам в определённый период времени. Найти эту аналитическую форму можно в главном меню в папке **CRM**

![](image.png)

Форма состоит из трех вкладок:

- **Отгрузки по новым контрагентам** - категория, по которой можно сгруппировать данные по товарной группе.
- **Менеджер** – ответственный за работу с клиентом.
- **Сумма** – общая стоимость отгрузки.
- **Объём, кг** – физический вес отгруженного товара.

**"Отгрузка по новым котрагентам с расшифровкой"** - содержится та же информация, что и в основной вкладке, а так же здесь можно узнать информацию о контрагент еи о дате его создания.

![](image-1.png)

**"Графики"** - В этой вкладке представлен аналитический отчёт по отгрузкам новым клиентам с упором на визуализацию данных в виде графиков и сводных показателей.

![](image-2.png)
