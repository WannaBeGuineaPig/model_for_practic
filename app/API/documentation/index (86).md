# Штатное расписание

## Описание

**Штатное расписание** — нормативный документ предприятия, оформляющий структуру, штатный состав и численность плановых штатных единиц с указанием размера оплаты труда в зависимости от занимаемой должности. Перед формированием штатного расписания должен быть заполнен справочник [«Штатные единицы»](../Штатные единицы/index.md).

## Создание штатного расписания

Для создания нового штатного расписание необходимо в реестре штатного расписания нажать кнопку "Создать". Откроется новая табличная форма:

![Новая форма](media/newForm.png)

В данной форме заполняется следующая информация:

* В группе столбцов **"Штатная единица"** в столбце **"Код"** - указывается код штатной единицы. Столбцы "Подразделения", "Должность", "Количество" заполняется автоматически;

* В столбце **"Грейд"** из выпадающего списка выбирается грейд для текущей штатной единицы.

Остальные столбцы заполняются автоматически в зависимости от заполнения документа [«Штатные единицы»](../Штатные единицы/index.md).

В таблице **"Позиции"** указывается следующая информация:

* ID позиции
* Грейд;
* Оклад по грейду;
* Планируемая зарплата на предприятии по текущей штатной единице;
* Группа столбцов "Обзор ЗП" заполняется согласно справочника [Обзор заработных плат и компенсаций](../Обзор заработных плат и компенсаций/index.md)
