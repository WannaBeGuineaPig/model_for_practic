# Ресурс обучения

В справочник вносится информация о ресурсах обучения.

Для открытия документа необходимо в Меню в поисковой строке ввести - "**Ресурс обучения**. 
У пользователя должены быть настроена роль **StaffTraining_Admin**

Для создания нового ресурса обучения в реестре нажмите кнопку "Создать"

Заполните поле "Наименование".

Нажмите "Сохранить и Закрыть"
