# Совещание

## Описание

В данной статье описан порядок работы по организации совещаний в системе.

## Создание нового совещания

Создать совещание возможно двумя способами: через [форму](#работа_с_формой) либо через [виджет](#работа_через_виджет) 

### Работа с формой

Для открытия документа необходимо в Меню в поисковой строке ввести - Совещания. И в открывшемся реестре нажать кнопку "Создать". В блоке **"Регистрационные данные совещания"**, заполняются тема, дата, время и место проведения совещания. В блоке **"Участники"** заполняются сотрудники, которые будут присутствовать на совещании. При необходимости возможно добавить вложения-документы к совещанию.

![Форма](media/docInit.png)

### Работа через виджет

На главной странице в левой панели необходимо нажать на иконку календаря.

![Виджет](media/vidjet.png)

В появившемся окне можно просмотреть, организовать новое совещание, отредактировать и удалить совещания, а также создать [протокол](../Протокол совещания/index.md) по проведенному совещанию. В календаре по умолчанию указан текущий месяц.Чтобы организовать  новое совещание, необходимо выбрать день, время и нажать на кнопку **"Создать совещание"**.

![Новое совещание](media/newMeet.png)

В появившемся окне в блоке **"Регистрационные данные совещания"** указываются тема, дата, время и место проведения совещания. В поле "Участники" указываются сотрудники, которые будут присутствовать на совещании. Для добавления участников необходимо нажать на кнопку **"+Сотрудник"**, в появившемся окне выбрать одного или нескольких сотрудников, поставив галочки, и нажать кнопку добавить. При необходимости возможно добавить вложения-документы к совещанию.

![Новое совещание](media/newMeeting.png)

После создания нового совещания необходимо нажать на кнопку **"Продолжить"**. Новое совещание будет отображено в календаре, а участникам автоматически будет направлено приглашение.

![Приглашение](media/meeting.png)

![Почта](media/mail.png)

Если необходимо отредактировать либо удалить совещание, необходимо выбрать совещание и нажать соответствующую кнопку в виджете.

При отмене совещания всем участникам будет направлено уведомление.

![Отмена совещания](media/canceledMeet.png)
