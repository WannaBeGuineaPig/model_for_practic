# Исполнение годовой заявки на потребность по договорам

Аналитическая форма “Исполнение годовой заявки на потребность по договорам” предназначена для анализа данных по годовым заявкам на потребность. Найти ее можно в главном меню системы.

![](image.png)

Форма выглядит следующим образом:

![](image-1.png)

В ней отображается следующая информация: 

* Организация, которая создала Годовую заявку на потребность;

* Информация о Годовой заявке на потребность: ссылка, наименование материала и его код, ЕИ, Количество, Цена, Сумма;

* Ссылки к Договорам Годовых заявок на потребность;

* Ссылки к Доп.документам Годовых заявок на потребность, а также Количество, Цена/Сумма без НДС, Цена/Сумма с НДС. 

После открытия аналит.формы выберите Период, за который хотите загрузить Годовые заявки на потребность. Для этого выберите соответствующую позицию из выпадающего списка и нажмите кнопку “Сформировать”:

![](image-2.png)

В табличной части отобразятся Годовые заявки на потребность за выбранный Период:

![](image-3.png)

При нажатии на годовую заявку на потребность, договор или документ, откроются соответствующие документы

![](image-4.png)
