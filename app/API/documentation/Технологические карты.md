# Технологические карты

Данный документ предназначен для отражения режима изготовления готовой продукции: какие операции выполняются при изготовлении ГП, какое сырье используется в той или иной операции. 

!!! Примечание

    Технологическую карту можно создать с Составом изделия или без Состава изделия

Чтобы создать технологическую карту, необходимо в главном меню перейти в **"Технологические карты"**. 

![](image.png)

Откроется реестр имеющихся документов. 

![](image-1.png)

Для того, чтобы открыть старый документ - кликните по нему, для того, чтобы создать новый - нажмите по соответствующей кнопке ![](image-2.png)

Откроется форма создания нового документа, состоящая из шапки документа и трех табличных частей.

![](image-3.png)

В шапку документа автоматически заполнятся такие поля, как:

- **Дата** - заполняется текущей датой, есть возможность редактирования
- **Статус** - заполняется значением "создано", для того, чтобы в плане производства сделать разузловку по технологической карте, нужно поставить статус **"Проверено"**
- **Период с** - заполняется текущей датой, есть возможность редактирования
- **Номер** - заполняется автоматически при сохранении документа или прописывается вручную;

Поля, обязательные для заполнения:

![](image-4.png)

* **Изделие** - заполняется выбором из выпадающего списка. В изделие можно перейти по кнопке ![](image-5.png)
* **Состав изделия** – заполняется автоматически при выборе изделия если в системе существует только один состав изделия, если у изделия в системе составов больше, то нужно выбрать один из выпадающего списка. В состав изделия можно перейти по кнопке (кнопка ![](4-1.jpg)). Подробнее о заполнении Состава изделия [читать тут](Lexema-ERP/../../../Управление%20производством/Составы%20изделий/Составы%20изделий.md). 

* **Шифр модификации** – заполняется автоматически после выбора Состава изделия;

* **Выход готовой продукции (количество), ЕИ** – заполняется автоматически после выбора Состава изделия.

Есть возможность выбрать **Базовую ТК** (если есть) - выбирается из списка. При нажатии на ![](6.jpg) осуществится переход в Базовую ТК:

При нажатии на кнопку **“Заполнить”** заполнится таблица значениями из Базовой ТК:

![](image-6.png)

Текстовые поля, которые заполняются по необходимости:

* **Контрагент** – заполняется выбором из списка
* **Примечание** - для заполнения дополнительной информации
* **Заявка от клиента** - заполняется выбором из выпадающего списка

Таблица “Список операции” также заполняется по кнопке ![](9-1.jpg). В новой строке выберите операцию из списка. Автоматически заполнится и наименование Содержания. 

![](image-7.png)

* **Операция** - определенный этап производственного процесса, который может подлежать контролю для подтверждения соответствия установленным нормам. Выбирается из выпадающего списка.
* **Требуется ОТК** - чекбокс требующий пройти **Отдел технического контроля**.
* **Исполнение** - при нажатии открывается окно должностями, разрядами сотрудников и временем, которое они затратят на обработку данного изделия.

![](image-8.png)

!!! Примечание
    Чтобы время обработки было в минутах/часах, нужно отредактировать константу units_of_time: в ней прописывается JSON файл с секундами, минутами, часами и т.д.  Используется та единица измерения, где стоит "byDefault":true.

* **Оборудование** - при нажатии открывается окно с оборудованием, нужным для данного этапа.

![](image-9.png)

В верхнем правом углу находится табличная часть с характеристиками изделия, которые указываются в документе **Номенклатура**. По кнопке "Править" можно изменить характеристику.

![](image-10.png)

В нижнем правом углу находится табличная часть, состоящая из трех вкладок: Спецификация, параметры процесса и характеристики.

Вкладка **"Спецификация"** - это перечень и объем узлов и материалов, используемых на данной операции.  

Чтобы добавить спецификации выделите Операцию и нажмите на ![](image-11.png)  таблице “Спецификация”

!!! Примечание

    Так же спецификацию можно заполнить в Составе изделия

Вкладка **"Параметры процесса"** - это перечень параметров с допусками

Вкладка **"Характеристики"** - это список характеристик изделия, при которых будет подбирается данная операция

Сохраните документ по соответствующей кнопке. 

Чтобы посмотреть, имеются ли в документе расхождения, нажмите на кнопку ![](17-1.jpg) на панели управления, а так же проверка произойдет при сохранении документа.

![](18-1.jpg)

Сохраните документ с помощью кнопок в верхнем меню.
