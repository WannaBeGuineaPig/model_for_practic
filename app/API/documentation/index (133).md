# Переносы отпусков

## Описание

Переносы отпусков - форма в которой возможно не только просматривать информацию о уже перенесенных отпусках, но и создавать новые запросы на перенос отпусков.

## Работа с формой

Для открытия реестра необходимо перейти в Меню - Кадровый документооборот.Реестры - Переносы отпусков. Откроется реестр с переносами отпусков.

![Перенос](media/perenos.png)

Данные по переносу отпуска в реестре необходимо нажать "Создать".

В открывшейся форме необходимо выбрать подразделение из выпадающего списка. 

![Подразделение](media/podraz.png)

Для добавления нового переноса в табличной части необходимо нажать на кнопку "+".

![Добавление](media/add.png)

В столбце **"Сотрудник"** из выпадающего списка выбирается сотрудник предприятия и на какой период согласно графика, оформляется перенос отпуска. 

![Сотрудник](media/sotrudnik.png)

Группа столбцов **"Отпуск"** заполняется автоматически по данным из графика отпусков. 

![Отпуск](media/otpusk.png)

В группе столбцов **"Перенос отпуска"** заполняется новая дата начала отпуска, количество дней и основание для переноса отпуска.

![Перенос](media/perenos2.png)

Должность, подразделение и родительское подразделение заполняются автоматически. **Примечание** указывается при необходимости. После этого необходимо проставить галочку напротив поля "Перенос утвержден".

![Утвержден](media/utver.png)
