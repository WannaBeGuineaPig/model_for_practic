# Сотрудники

При выгрузке данных из 1С набор данных для загрузки в СЭД определяется  как список сотрудников, выбранных из сущности Catalog_Сотрудники  и для которых еще нет записи в таблице соответствия по коду 1С.<br>
В поля подразделения и должность загружаются последние данные по сотруднику с учетом данных из регистра «InformationRegister_КадроваяИсторияСотрудников».<br>
* Справочник сотрудников	        - Catalog_Сотрудники <br>
* Кадровые данные сотрудников 	    - InformationRegister_КадроваяИсторияСотрудников<br>
