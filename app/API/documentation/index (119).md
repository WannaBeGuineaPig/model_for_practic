# Личный кабинет сотрудника

Личный кабинет сотрудника включает в себя личных данных сотрудника, стаж работы, «зарплатную ведомость», оставшиеся дни отпуска, отправленные документы и локальные нормативные акты.

Чтобы открыть форму, необходимо выбрать пункт «Личный кабинет» в плиточном меню.

![Меню профиля](./media/MenuProfil.png)

Откроются личные данные текущего сотрудника.

## Персональные данные

На данной вкладке отображаются личные данные сотрудника. 

![Персональные данные](screens/screen1.png)

Если персональные данные изменились, сотрудник может направить запрос на изменение. Для этого необходимо нажать на кнопку **"Запрос изменение данных"**.

При нажатии на кнопку **"Другие места работы"** в табличной части будут отображены филиалы холдинга, в которой сотрудник работает по совместительству.

![Сорг](screens/screen2.png)

При нажатии на кнопку **"Изменение телефона/эл.почты"** открывается всплывающее окно для редактирования номера телефона / электронной почты. После изменения данных, необходимо нажать на кнопку "Сохранить".

![Изменение](media/readPhone.png)

Доступ к данной кнопке имеют пользователи с определенными ролями. Для настройки необходимо обратиться к администратору.

## Документы

На данной вкладке отображаются паспортные данные сотрудника, ИНН, СНИЛС, об образовании и военном билете.

Если данные по документам изменились, сотрудник может направить запрос на изменение. Для этого необходимо нажать на кнопку "Запрос изменение данных" в нужном разделе, например, "Данные паспорта".

![Запрос](screens/screen3.png)

После нажатия на кнопку откроется документ "Заявление от сотрудника" с выбранным подтипом. В блоке "Реквизиты документа" необходимо ввести данные, которые необходимо изменить и нажать кнопку «Заполнить шаблон» для отправки документа по маршруту.

![Персональные данные-заявление](./media/NewINN.png)

Более подробно про заполнение документа можно ознакомиться в разделе: ["Заявление от сотрудника"](../../../КЭДО/Заявления от сотрудников/)

![Персональные данные-расчетный лист](screens/screen4.png)

После клика по ссылке с наименованием "Расчетный лист" откроется печатная форма "Расчетный листок".

![Персональные данные-расчетный листок](./media/RaschList.png)

## Графики отпусков

На данной вкладке отображается график отпусков сотрудника по всем местам его работы. В форме отображаются данные согласно графика отпусков за текущий год с учетом всех переносов отпусков. Для просмотра графика отпусков по другим местам работы, необходимо выбрать организацию. Чтобы развернуть группу, нужно нажать на треугольник слева от поля с годом / в строке с годом. По умолчанию показывается график по текущему филиалу.

![alt text](screens/2024-12-02.png)

При очищении поля "Организация" показываются графики отпусков по всем местам его работы

![alt text](screens/screen6.png)

Сотрудник, обладающий определенными правами, имеет возможность создать заявление на отпуск, перенос или компенсацию в соответствии с его графиком отпусков. Для этого нужно выбрать необходимую строку с отпуском в графике и нажать на "Создать заявление"  и в выпадающем списке выбрать необходимый тип заявления.

![alt text](screens/screen7.png)

После выбора соответствующего действия, связанного с отпуском, например, создание заявления на отпуск, в новой вкладке появится форма подачи заявления на ежегодный отпуск, в которой автоматически будет заполнены реквизиты документа.

![alt text](media/image-6.png)

## Нормативные акты

На данной вкладке отображаются «Локальные нормативные документы», переданные сотруднику для ознакомления и ранее прочитанные им.

![Персональные данные-ЛНА](screens/screen9.png)

Столбец **"Открыть"** содержит код (гиперссылку) для перехода к основному документу. Чтобы перейти к соответствующему документу необходимо нажать на гиперссылку.

Столбец **Статус** предоставляет информацию о текущем состоянии маршрута каждого документа.

В столбце **"Вложение"** отображается информация о вложенном файле. Для просмотра вложенного файла, нужно нажать на значок вложения. Содержимое вложения откроется в правой части окна.

Столбцы **"Дата", "Номер", "Документ"** отображают соответствующую информацию, такую как дата создания документа, номер документа и тип документа.

В столбце **"Краткое содержание"** отображаются краткое содержание локального нормативного акта.

В столбце **"Организация"**  отображается в какой организации холдинга был создан документ.

## Мои замещения 

Форма "Мои замещения" необходимо для того чтобы указанный пользователь мог зайти под замещаемым пользователем. Это необходима чтобы сотрудник мог назначать заместителя на срок своего отсутвия. 

Ниже представлен пример, как выглядит фоpма "Мои замещения" 

![рисунок](screens/image.png) 

В выпадающем списке _**"Что хотите сделать?"**_ необходимо выбрать действие. Добавить заместителя или удалить уже существующего.

Для добавления сотрудника в список замещаемых необходимо после выбора действия из выпадающего списка в другом выпадающем списке под названием _**"Кто меня будет замещать?"**_ выбрать пользователя.

Так же можно выбрать на какой срок сотрудник будет замещать, для этого из выпадающего списка под названием _**"На какой срок?"**_ выбрать строку _"бессрочно"_ или _"на срок"_. В случае выбора строки _"на срок"_ появится поля где необходимо выбрать дату с какого числа замещение будет действовать, по какое число включительно. После чего необходимо подтвердить назначение замещения с помощью кнопки _"Подтвердить назначение прав замещения"_.

 Пример на скрине ниже:

![Рисунок](screens/screen13.png)

После чего в выпадающем спике _"Чьи права замещения удалить?"_ выбрать пользователя и нажать кнопку _"Подтвердить удаление прав замещения"_. Подробнее ознакомиться можно на скрине указанном ниже:
