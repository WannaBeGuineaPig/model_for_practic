# Годовая заявка на потребность

Годовой план потребности формируется на основании документов Годовая заявка на потребность. Годовая заявка на потребность предназначена для позиций, закупаемых на внутреннем рынке, включая данные по услугам.

Для работы с документом откройте реестр Годовой заявки на потребность. Для создания нового документа нажмите кнопку **"Создать"**.

![](image.png)

В новом документе автоматически заполняются поля: 

* **Дата** – по умолчанию заполняется текущей датой;

* **Инициатор** – заполняется сотрудником, создающим документ;

* **Подразделение** – подтягивается подразделение инициатора.

![](image-1.png)

* **Период** - поле, обязательное для заполнения. Если это поле не заполнить, то появится всплывающее окно с ошибкой:

![](image-2.png)

* **Номер** - можно заполнить вручную либо он заполнится автоматически при сохранении документа.

Поля **Вид потребности, Вид, Статус, Основание для закупки** не являются обязательными для заполнения:

![](image-3.png)

Для того, что бы заполнить табличную часть нажмите кнопку Добавить ![](image-4.png), чтобы появилась новая строка.

![](image-5.png)

В колонке Код/наименование материала выберите позицию из выпадающего списка или через обозреватель аналитик по кнопке ![](image-6.png).

Автоматически подтянутся такие данные как: Группа и код номенклатуры, Ед.измерения.

![](6.png)

Укажите Годовую цену:

![](8.png)

Выберите месяц и укажите количество номенклатуры, которое потребуется закупить в этом месяце. Автоматически рассчитается Сумма, которая затратится на покупку указанного количества: 

![](image-7.png)

Если необходимая номенклатура отсутствует в списке и в обозревателе аналитик, введите ее в поле Номенклатура(ручной ввод). Укажите ЕИ и Группу:

![](image-8.png)

Выберите месяц и укажите количество номенклатуры, которое потребуется закупить в этом месяце. Автоматически рассчитается Сумма, которая затратится на покупку указанного количества: 

![](image-9.png)

По кнопке **"Операции"** в Годовой заявке на потребность можно создать: 

1. оперативную заявку на потребность в товарно-материальных ценностях по выделенным позициям.

2. договор на услуги со спецификацией по выделенным позициям.

![](image-10.png)

## Создание заявки на потребность товарно-материальных ценностей 

В Годовой заявке на потребность выберите строки, по которым хотите создать Заявку на потребность. Для этого одновременно зажмите клавишу CTRL + левую кнопку мыши на нужных строках с материалами:

![](image-11.png)

После выделения строк нажмите на кнопку **Операции – Создать заявку на потребность**:

![](image-12.png)

Откроется окно с выбором месяца для оперативного создания Заявки на потребность. Выберите его из выпадающего списка, а затем нажмите **Сформировать**: Выбирать нужно тот месяц, который вы указали в табличной части.

![](image-13.png)

Откроется созданная Заявка на потребность. Выберите номенклатуру или создайте новую для номенклатуры ручного ввода. Так же можно выделить номенклатуру ручного ввода и нажать на кнопку **Создать номенклатуру**.

![](image-14.png)

Номенклатура с кодом появится в поле Код/наименование материала. 

![](image-15.png)

При необходимости измените количество. Сохраните и закройте документ.

При этом в Годовой заявке на потребность строки, по которым уже создана Заявка на потребность окрасятся в зеленый (если лимит годовой заявки исчерпан полностью), либо в желтый (если в годовой заявке по этой строке остался неиспользованный лимит).

![](image-16.png)

Если попытаться создать еще одну Заявку на потребность по строке, окрашенной зеленым, система выдаст предупреждение: 

![](19.png)

## Формирование договоров с поставщиками услуг

В Годовой заявке на потребность выберите строки, по которым необходимо сформировать договор. Для этого одновременно зажмите клавишу CTRL + левую кнопку мыши на нужных строках с услугами, затем нажмите на кнопку **Операции – Создать договор (для услуг)**

![](image-17.png)

В открывшемся окне выберите Контрагента из выпадающего списка или из Обозревателя аналитик:

![](22.png)

В системе сформируется Договор со спецификацией. Строки, с которыми создался договор и спецификация, окрашиваются зеленым цветом:

![](image-18.png)

При нажатии на кнопку **"Связки"** ![](image-19.png) можно посмотреть все связанные документы, а так же перейти к ним после нажатия на наименование документаы

![](image-20.png)
